1. 개요

- 분할정복법은 항상 문제를 절반씩으로 나누는 분할 정복 알고리즘
- 일반적인 재귀 호출과 다른 점은 문제를 한 조각과 나머니 전체로 나누는 대신 거의 같은 크기의 부분 문제로 나눔
- 일단 제일 큰 사이즈의 문제를 넣고 그것을 작게 재귀형태로 분할한다.



2. 어떻게 푸는가

-> 대부분의 문제들은 이차원 리스트를 주고 그것을 분할하는 문제
-> 좌표간의 관계가 중요함
-> 우선 리턴할 사이즈를 정하고 리턴할 사이즈가 나올때까지 분할한다. size//3, size//2 ...

-> 리턴할 사이즈를 만족하면 그 좌표부터 로직을 수행한다.
-> 분할된 영역의 첫 좌표를 찾는것이 중요*****
